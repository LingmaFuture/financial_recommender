接口概述

本系统提供RESTful API接口，支持JSON格式的数据交互。所有请求都需要包含适当的Content-Type头部。

POST /recommend

根据用户信息生成金融产品推荐。

请求参数

参数名 类型 必填 说明
AGE Number 是 用户年龄
GENDER String 是 性别
MARITAL String 是 婚姻状况
HAVE_CHILD String 是 是否有子女 (Y/N)
EDU_LEVEL String 是 教育水平

示例请求

{
    "AGE": 35,
    "GENDER": "M",
    "MARITAL": "MARRIED",
    "HAVE_CHILD": "Y",
    "EDU_LEVEL": "BACHELOR"
}

示例响应

{
    "recommendations": [
        {
            "product_id": "FP001",
            "name": "稳健理财产品A",
            "score": 0.95,
            "description": "适合稳健型投资者的理财产品"
        }
    ]
}

POST /chat

智能客服对话接口。

请求参数

参数名 类型 必填 说明
question String 是 用户问题

示例请求

{
    "question": "请介绍一下你们的理财产品"
}

示例响应

{
    "answer": "我们提供多种类型的理财产品，包括..."
}
